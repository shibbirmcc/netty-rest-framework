# rest-netty-framework

Lightweight, easy-to-scale REST API framework built on Netty for high-performance microservices.

## Features
- Built on Netty (non-blocking I/O) for high concurrency. :contentReference[oaicite:2]{index=2}  
- Easy to define endpoints, middleware, JSON serialization.  
- Pluggable routing, handlers, filters, metrics.  
- Designed for containerisation and horizontal scaling.

## Getting Started
1. Clone the repo.  
2. Build: `./gradlew clean build` (or Maven)  
3. Run sample: `java -jar rest-netty-framework.jar`  
4. Access: `http://localhost:8080/health` etc.
